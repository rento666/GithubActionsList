## 功能说明

- 自动登录小白游戏网 (xbgame.net)
- 自动完成每日签到
- 查询用户信息：等级、积分、余额、任务进度
- 签到结果包含：获得积分、连续签到天数、累计签到积分
- 支持邮件通知
- 运行结果自动归档到 keep-alive 分支

## 运行周期

| 触发方式 | 时间 | 说明 |
|:---------|:-----|:-----|
| 定时触发 | 每天北京时间 08:40 | `cron: '40 0 * * *'` |
| 手动触发 | 随时 | Actions → Run workflow |
| Push 触发 | 推送时 | 修改 `XBGame/**` 或 `utils/**` 路径 |

## 环境变量

### 必填

| 名称 | 类型 | 说明 |
|:-----|:-----|:-----|
| `XB_USERNAME` | Secret | 小白游戏网用户名 |
| `XB_PASSWORD` | Secret | 小白游戏网密码 |

### 可选

| 名称 | 类型 | 默认值 | 说明 |
|:-----|:-----|:-------|:-----|
| `XB_NOTIFY_SUCCESS` | Variable | false | 签到成功时发送邮件 |
| `XB_NOTIFY_ERRORS` | Variable | false | 签到失败时发送邮件 |

## 运行开关

| 变量名 | 默认值 | 说明 |
|:-------|:-------|:-----|
| `ENABLE_ALL` | 启用 | 全局总开关 |
| `ENABLE_XBGAME` | 启用 | 本脚本开关 |

## Workflow 执行流程

```
1. 检查运行开关 → 是否跳过
2. Checkout 仓库
3. 设置 Node.js 22
4. 安装依赖 (axios, nodemailer)
5. 执行签到脚本 → 输出到 /tmp/script-output.txt
   ├── 登录获取 JWT Token
   ├── 查询用户信息（等级、积分、余额）
   ├── 执行每日签到
   └── 输出签到结果
6. 保存结果到 keep-alive 分支
7. 检查脚本执行结果 → 标记成功/失败
```

## API 接口

| 接口 | 方法 | 说明 |
|:-----|:-----|:-----|
| `/wp-json/jwt-auth/v1/token` | POST | JWT 登录认证 |
| `/wp-json/b2/v1/userMission` | POST | 执行每日签到 |

## 错误处理

- 登录失败：检查用户名密码是否正确
- Token 获取失败：密码可能已变更，更新 Secret
- 签到接口异常：查看 Actions 日志中的详细错误
